# auto-tetris

Play Tetris game automatically!

## Features
* AI for Tetris
* Simple terminal user interface (TUI)

## Build and Run
To build `auto-tetris`, run `make` inside the directory where you have the source.
```shell
make
```

To play the game, run the executable 'tetris'.
```shell
./tetris
```

Key mapping:
* space: Pause toggle
* Q: Quit the game
